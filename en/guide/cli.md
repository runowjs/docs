# CLI

### Initialization

### `runow`

Can be used in any directory,  `create-runow` is aliases for `runow`, Both are acceptable.

#### Started

```bash
runow
```

#### Options

| Options                 |                               |
|-------------------------|-------------------------------|
| `-t, --template [name]` | Specify template name         |
| `-h, --help`            | Display available CLI options |
